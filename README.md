## 📌 Featured Projects

### 🔹 Data Science & Machine Learning

- [**ECG Anomaly Detection — LSTM Autoencoder (PyTorch)**](https://github.com/tkaushik015/ECG-Anomaly-Detection) – Trained a **2-layer LSTM Autoencoder in PyTorch** on ECG5000 (5,000 heartbeat sequences) to detect abnormal cardiac rhythms via reconstruction error. Trained on normal beats only, selected anomaly threshold by maximizing F1 on a held-out eval set — achieving **0.89 F1 and 0.93 ROC-AUC**. Ran 6 MLflow experiments across latent dimensions (16/32/64), identifying latent dim 32 with 0.1 dropout as optimal (**14% lower false positive rate**). Deployed as a FastAPI endpoint on GCP Cloud Run.  
  `Python · PyTorch · MLflow · SHAP · FastAPI · GCP Cloud Run · scikit-learn`

- [**GenFinance: Multi-Agent RAG for Financial Document Analysis**](https://github.com/tkaushik015/GenFinance-Multi-Agent-RAG-System-for-Financial-Document-Analysis) – Built a **multi-agent RAG pipeline** using LlamaIndex, CrewAI, and AutoGen over SEC filings and earnings call transcripts, cutting financial document analysis from ~3 hours to **under 25 minutes per report**. Benchmarked GPT-4o and Claude Sonnet on 50 chart interpretation tasks using DeepEval — Claude Sonnet delivered **22% lower hallucination rate at 40% lower cost**, driving final model selection.  
  `Python · LlamaIndex · CrewAI · AutoGen · DeepEval · GPT-4o · Claude Sonnet`

- [**BabyWeight Predictor: End-to-End MLOps on Vertex AI and Kubeflow**](https://github.com/tkaushik015/CloudML-Engine-End-to-End-MLOps-Pipeline-on-GCP-Vertex-AI-Kubeflow-) – Compared BQML and AutoML on GCP Vertex AI Pipelines using BigQuery Natality data — AutoML reduced RMSE by **18% over the BQML baseline**. Deployed winning model with blue/green strategy and lineage tracking in Vertex AI Model Registry. Automated retraining with Cloud Build CI/CD and Kubeflow Pipelines with Vertex AI Model Monitoring for data drift detection — cutting deployment cycle time from **~4 hours to under 35 minutes**.  
  `Python · GCP Vertex AI · Kubeflow · BigQuery ML · Cloud Build · Terraform · Docker`
